```tsx
// Button.tsx
export function Button({ variant = 'primary', children, ...props }) {
  const baseClasses = "px-6 py-3 rounded-lg font-semibold transition-all duration-200";
  const variants = {
    primary: "bg-primary text-white shadow-md hover:bg-primary/90 hover:shadow-lg",
    secondary: "bg-white text-primary border border-primary shadow-sm hover:bg-primary/5",
    text: "text-primary hover:text-primary/80 underline-offset-4 hover:underline"
  };
  
  return (
    <button className={`${baseClasses} ${variants[variant]}`} {...props}>
      {children}
    </button>
  );
}
```

## 实施策略

### 渐进式更新
1. **保持向后兼容**：新旧样式可以共存一段时间
2. **分页面更新**：一次只更新一个页面，减少风险
3. **功能开关**：可以使用功能标志控制新样式

### 代码管理
1. **分支策略**：为UI改进创建专门的分支
2. **代码审查**：确保设计一致性
3. **文档更新**：更新组件使用文档

### 测试策略
1. **视觉回归测试**：使用工具如Chromatic
2. **用户测试**：邀请真实用户测试新设计
3. **A/B测试**：对比新旧设计的用户行为
